GHS Media Project 2020

How does freedom and liberty stem from inequality?

This Project is based upon Angela Davis's writing. I will specifically draw upon The Meaning of Emancipation According to Black Women in order to show prison systems have given our culture an ideation of what freedom and inequality is. In terms of the technical aspects of this project, I'm using Angular for my web framework, Angular Material to define standard components and themes, and finally Github Pages in order to host this website.
